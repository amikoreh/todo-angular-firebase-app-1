todo-angular-app-1
================
#### Todo app demo with Firebase and Angular
Back-end: Firebase back-end with OAuth authentication
Front-end: Angular, AngularFire2, and Typescript

Cloned from: https://github.com/r-park/todo-angular-firebase

#### Application Requires/Configuration:
* Firebase account
* Angular @4.4.4
* Angular CLI @1.2.0
* angularfire2@4.0.0-rc0
* Firebase 4.4.0
* rxjs@5.4.3
* Sass 3.5.1
* typescript@2.5.3
* npm 5.4.2
* Node 8.6.0
* webpack 2.4.1

##### Build + Deploy:
```shell
$ npm install
$ npm start
```

* Build, Test, + Deploy
```shell
$ npm run lint
$ npm run build
$ firebase login
$ firebase use default
$ firebase deploy
```

* Other required npm libraries
```shell
$ npm install -g firebase-tools
$ npm install -g @angular/cli
```
